- **Crystal Band Gap Prediction** A machine learning–based project focused on predicting material band gap values using scientific crystal structure data and hybrid GNN-ML architectures.  
  - **Focus:** Benchmarking traditional ML models against graph-based deep learning approaches for material informatics.  
  - **Status:** Core framework for the JCIM publication.  
  🔗 [GitHub Repository](https://github.com/YZE-Crystal/bandgap-prediction)


- **CEMoE: Crystal-Entropy Mixture of Experts** An advanced research project implementing a scalable pipeline for predicting electronic properties (**Band Gap & VBM**) of crystalline materials.  
  - **Innovation:** Integrates **Graph Neural Networks (GNN)** with an **Entropy-Aware Gating Network** to route latent embeddings into specialized expert layers.  
  - **Scale:** Processed ~66k crystal structures from CIF files into graph objects using PyTorch Geometric.  
  🔗 [GitHub Repository](https://github.com/YZE-Crystal/CEMoE)
